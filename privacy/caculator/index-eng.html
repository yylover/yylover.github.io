Life Calculator Privacy Policy

Life Calculator Privacy Policy

Last Updated: February 14, 2026

1. Introduction

Life Calculator (hereinafter referred to as "this App") is committed to protecting user privacy and personal information. This Privacy Policy aims to explain how this App collects, uses, stores, and protects user information, as well as how users can manage their own information.

2. Data Collection

1. Information Not Collected

This App does not collect the following types of personal information: - Personal contact information such as name, email address, or phone number - Geolocation information - Device identifiers - Browsing history or usage habits - Photos, videos, or other media files - Health or medical information (even when using health calculation features)

2. Locally Stored Information

This App stores the following information locally: - Calculation history: Including user's calculation inputs, results, and notes - App settings: Including theme preferences, sound settings, vibration feedback settings, etc. - Archive data: Including calculation processes and results saved by users - Exchange rate cache: Including recently obtained exchange rate data (for offline use)

3. Network Requests

This App uses network connections only in the following cases: - Exchange rate conversion: Obtaining real-time exchange rate data - App update check: (If implemented) Checking if there are available updates for the App

3. Data Usage

1. Local Data Usage

This App only uses stored information locally: - Calculation history: For displaying history records, supporting archive functionality, and segmented calculations - App settings: For personalizing user experience - Archive data: For saving and restoring calculation processes - Exchange rate cache: For basic conversions when offline

2. Network Data Usage

This App uses network data only for: - Obtaining exchange rates: Getting real-time exchange rate data from public exchange rate APIs - App updates: (If implemented) Checking App update status

4. Data Storage

1. Storage Methods

This App uses the following methods to store data: - SwiftData: Apple's officially recommended local data storage framework - UserDefaults: Storing App settings and preferences - Cache: Storing temporary data such as exchange rate cache

2. Data Security

This App takes the following measures to protect user data: - Local storage: All data is only stored locally on the user's device and not uploaded to any server - Data isolation: App data is isolated from other apps, following iOS sandbox mechanisms - Encrypted storage: Following iOS system data protection mechanisms

3. Data Deletion

Users can delete data through the following methods: - Clear history: Manually delete calculation history within the App - Delete archives: Manually delete saved archives within the App - Uninstall App: Uninstalling the App will delete all related data

5. Third-Party Services

1. Exchange Rate API

This App uses public exchange rate APIs to obtain real-time exchange rate data. These APIs may collect anonymous usage data, such as request IP addresses, but do not collect personally identifiable information.

2. Other Third-Party Services

This App does not use the following third-party services: - Analytics services (such as Google Analytics) - Advertising services - Social media integration - Crash reporting services

6. Children's Privacy

This App is suitable for users of all ages and does not collect any personal information from children. This App complies with the requirements of COPPA (Children's Online Privacy Protection Act).

7. Privacy Policy Updates

This Privacy Policy may be updated from time to time. When we make significant changes, we will notify users within the App and update the "Last Updated" date of this policy.

8. User Rights

Users have the right to: - Access their data stored on the device - Modify their calculation records and archives - Delete their calculation records and archives - Control App settings and preferences

9. Contact Us

If you have any questions or suggestions about this Privacy Policy, please contact us through the following methods:

  • Email: quickyangjob@gmail.com
  • In-App feedback: Through the "Contact Us" feature in the settings page

10. Applicable Scope

This Privacy Policy only applies to the Life Calculator App and does not apply to other apps or websites, even if they are associated with this App.

11. Legal Compliance

This App complies with the following laws and regulations: - China Cybersecurity Law - Apple App Store Review Guidelines - General Data Protection Regulation (GDPR) (if applicable) - Children's Online Privacy Protection Act (COPPA)

Statement: This Privacy Policy aims to protect user privacy and ensure users understand how this App processes data. This App is committed to providing safe and reliable calculation tool services.
